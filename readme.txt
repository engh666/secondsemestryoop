В файле algorithms.h находятся все компоненты используемые в программе, такие как:
Main class of adress,  class database, Sl circular, AVL tree, B tree, Hash Table with Linear Probing, Hash Table chaining
Insertion sort, Heap sort, Merge sort, Count sort.

Интерфейс программы реализован на QT, он позволяет:
1. Добавлять фрагмент кода с указанными параметрами
2. Добавлять случайный фрагмент кода
3. Сортировать фрагменты кода по одному из параметров (метод, создатель) и
отслеживать время выполнения сортировок для сравнения производительности (Insertion sort, Heap sort, Merge sort, Count sort)
4. Добавлять/удалять фрагменты кода из списка(SL circular), проверять вхождения фрагмента кода в список, вычислять обьеденения, вычитание, семетричное вычитание и пересечение
5. Добавлять/удалять фрагменты кода из Hash Table with Linear Probing, Hash Table chaining а также просматривать соответсвующие ключи и их фрагменты кода
6. Добавлять/удалять фрагменты кода в AVL tree, B tree, а также просматривать их
